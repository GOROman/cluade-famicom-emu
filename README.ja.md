# claude-famicom-emu

Webブラウザで動くファミコン(NES)エミュレータ。コアは C++ で書き、Emscripten で WebAssembly にコンパイルしています。その上に「ハードウェアで遊ぶ」レイヤーを載せました——クリックで断線させられる60ピンのカードエッジ、傾けて半挿しを再現できるカセット、任意のピンにプローブを当てられるオシロスコープ。

**▶ 遊ぶ: https://goroman.github.io/cluade-famicom-emu/**

「ROMを開く」から手持ちの .NES ファイル(iNES形式)を読み込むと起動します。PC・Android Chrome 対応。

## 機能

### エミュレーションコア (C++ / WASM)
- **6502 CPU** — 全公式命令+主要な非公式命令。nestest.nes の全8991ステップをリファレンスログとサイクル数まで完全一致で検証済み。
- **PPU** — サイクル精度のスキャンライン処理、Loopyスクロール、スプライト0ヒット、スプライトオーバーフロー。
- **APU** — 矩形波×2 / 三角波 / ノイズ / DPCM を AudioWorklet で再生(非HTTPS環境では ScriptProcessor に自動フォールバック)。
- **マッパー** — 0 (NROM) / 1 (MMC1) / 2 (UxROM) / 3 (CNROM、64KBオーバーサイズ含む) / 4 (MMC3、スキャンラインIRQ)。
- 旧形式の iNES ヘッダ(末尾に `DiskDude!` 等のゴミが入ったもの)にも対応。
- バッテリーバックアップ(SRAM)は localStorage に自動保存。

### カートリッジ端子シミュレーション(「端子」モード・起動時オン)
- **60ピンのカードエッジ**を実機のピンアサインどおりに描画。ピンをクリックすると接触不良になります。
- 断線の影響は物理的に再現: アドレス線が欠けるとCPUが別の場所を読んで暴走、データ線が欠けるとオープンバス値に化け、CIRAM系が切れるとネームテーブルが全滅、SOUNDピンが切れると無音(ファミコンの音声はカセットを経由するため)、電源はピン1/16(GND)と30/31(+5V)で冗長。
- **カセットを傾ける** — スライダー(±6°、0.1°刻み)またはドラッグで。浮いた側の端子から接触が不安定になり、やがて断線します。まさに半挿しの挙動です。
- **息(フーフー)💨** — 接触不良が65%の確率で復活しますが、良好なPPU側ピンが10%の確率で湿気にやられます。SEとリセット付き(お作法どおり)。
- **挿し直す** — 全ピン復旧+傾き0°+リセット。

### カセット活線挿抜(バグ技)
**リセット**ボタンは実機と同じくワークRAMを保持します。**カセット入替**は*リセットを一切かけずに*新しいROMを読み込むので、電源を入れたままの差し替え裏技(スーパーマリオ→テニス→スーパーマリオで9-1)がそのまま成立します。ダイアログでは**PRG-ROMとCHR-ROMを別々のカセットから**組み合わせた「ニコイチカセット」も作れますし、**URLから**ROMを直接取得することもできます。

### デバッグ機能(DEBUGボタン / Dキー)
- **CHR (CGROM) ビューア** — パターンテーブルを現在のPPUパレットで着色表示。クリックでBG/スプライトパレットを切替。端子の接触不良もここに反映されます。
- **オシロスコープ** — 任意のコネクタピン(および /NMI・APU /IRQ・MAPPER /IRQ のテストポイント)にホバーするとプローブが当たります。コアがCPUサイクルごとにサンプリングし、60kHz未満ではリアルタイムのストリップチャート表示に切り替わります。
- **CPUレジスタ+逆アセンブル** — PC/A/X/Y/SP/P とフラグ、フレームカウンタ、そしてPCからの12命令ライブ逆アセンブル。
- **APU** — 6本の波形スコープ(SQ1/SQ2/TRI/NOI/DMC/MIX、ラベルクリックでチャンネルミュート)と $4000–$4017 のレジスタダンプ。
- **WRAMダンプ** — バイトをダブルクリックで編集可能。変化したバイトは光り、常時変化するバイト(タイマー・乱数)は灰色表示。スーパーマリオを読み込んでいるときは、ホバーで [SMBDIS.ASM](https://gist.github.com/1wErt3r/4048722) 由来の変数名と解説が出ます。

### 可変クロック
コネクタ下のスライダーとHz入力ボックスで、**1 Hz から定格 1.789773 MHz** まで(対数)。メインループはCPUサイクル単位で歩進するので、1桁Hzでも本当に1サイクルずつ動きます——ラスタがフレームを横切っていく様子が見えます。音程もクロックに追従します。

### TAS再生
TASボタンから FCEUX の **.fm2** ムービーを読み込みます。FCEUX方式のRAMパターンで電源投入し、ムービーの入力でフレーム単位に厳密実行。ムービー中のソフトリセット/電源コマンドも解釈します。

### その他
- USBゲームパッド(Gamepad API)、Android用タッチパッド、フルスクリーン(F)。
- UIは英語・日本語・中国語。既定ではブラウザのロケールに追従。
- **XEVIOUS判定** — 隠し機能の診断ツール。PRG/CHRのCRC32を正規ダンプと照合し、CHRが不一致ならダンパーのどのアドレス線/データ線が結線ミスかを割り出します。

## URLパラメータ

`https://goroman.github.io/cluade-famicom-emu/?rom=<URL>&debug=1&pin=0`

| パラメータ | 効果 |
|-----------|------|
| `rom=<URL>` | .NESファイルをURLから取得して起動(取得先のCORS許可が必要。GitHub rawは可) |
| `debug=1` | DEBUGパネルを開いた状態で起動 |
| `pin=0` / `pin=1` | 端子パネルの非表示/表示(既定は表示) |
| `clock=<Hz>` | クロック周波数(1〜1789773) |
| `tilt=<度>` | カセットの傾き(±6) |
| `break=25,29` | 指定ピンを断線状態で起動 |
| `mute=1` | ミュートで起動 |
| `lang=en/ja/zh/auto` | 表示言語 |

## 操作

| NES | キーボード | タッチ | ゲームパッド |
|-----|-----------|--------|------------|
| 十字キー | 矢印キー | 画面下パッド | 十字キー / 左スティック |
| A | X | A | 右ボタン |
| B | Z | B | 下ボタン |
| Start | Enter | START | Start |
| Select | Shift | SELECT | Select |

ホットキー: **F** フルスクリーン · **R** リセット(押している間はリセット状態) · **D** デバッグパネル

## ビルド

Emscripten (`emcc`) が必要です。

```sh
./build.sh   # → web/nes.js + web/nes.wasm を生成し、index.html にキャッシュバスト用の版数を刻印
```

## ローカル実行

```sh
cd web
python3 -m http.server 8765 --bind 0.0.0.0
```

- PC: http://localhost:8765/
- Android: 同一ネットワークから `http://<ホストのIP>:8765/`

## デプロイ (GitHub Pages)

`web/` を `gh-pages` ブランチとして配信しています。

```sh
./build.sh
git add -A && git commit -m "..."
git push
git subtree push --prefix web origin gh-pages
```

## 構成

```
core/    C++ エミュレータコア
  cpu.cpp        6502
  ppu.cpp        PPU(パレット、端子故障、NMIライン)
  apu.cpp        APU + チャンネル別スコープバッファ
  cartridge.cpp  iNESローダ + マッパー 0-4
  nes.cpp        バス、60ピン故障モデル、オシロプローブ、WASM C API
web/     フロントエンド (index.html / main.js / i18n.js / audio-worklet.js) + WASM出力
build.sh Emscripten ビルド + バージョン刻印
```

## テスト

- **CPU**: nestest.nes をリファレンスログと突き合わせるネイティブハーネス。
- **マッパー・ヘッダ・端子故障**: 合成ROMによるユニットテスト(ネイティブビルド)。

ROMファイルはこのリポジトリに含まれていません。手持ちの .NES ファイルをご利用ください。
